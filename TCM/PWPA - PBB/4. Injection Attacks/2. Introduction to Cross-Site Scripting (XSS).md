
## **What is XSS?**

- **XSS** = _Injecting malicious JavaScript into web pages that other users see_
    
- **Impact**: Steal cookies, session hijacking, defacement, phishing
    
- **OWASP**: #3 in Top 10 (Injection family)
## **Three Main Types:**

### **1. Reflected XSS**

- **Description**: Malicious script reflected in response (search, error messages)
    
- **Example**: `site.com/search?q=<script>alert(1)</script>`
    
- **Test**: `<script>alert(document.domain)</script>`
    
- **Delivery**: Send link to victim
### **2. Stored XSS**

- **Description**: Script stored on server (comments, profiles, forums)
    
- **Example**: Post comment: `<script>stealCookies()</script>`
    
- **Impact**: Affects all users who view page
    
- **Test**: Submit to all user-input fields
### **3. DOM-based XSS**

- **Description**: Client-side only, JavaScript modifies DOM
    
- **Example**: `document.write(location.hash)`
    
- **Test**: `#<script>alert(1)</script>`
    
- **Check**: URL fragments, `eval()`, `innerHTML`
