### **What is CSRF?**

- **Tricking user into making unwanted actions** on a website they're logged into
- **Example**: Make victim change their email/password without knowing
- **Impact**: Account takeover, data modification

#### **Simple Example :** 

```
You're logged into bank.com
        
You visit evil.com

evil.com has code that sends request to bank.com/transfer?to=hacker&amount=1000

Bank sees your logged-in session → Transfers $1000!
```
