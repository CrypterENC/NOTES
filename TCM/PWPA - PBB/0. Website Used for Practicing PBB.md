For `https://www.aikido.dev` and `https://app.aikido.dev`, the “main” domain is `aikido.dev`; both `www` and `app` are subdomains of it.​

## Parts of a domain

Using `https://www.aikido.dev` as an example:​

- **Protocol**: `https://` – how the browser talks to the server.
    
- **Top‑level domain (TLD)**: `.dev` – the extension at the end (.com, .org, .dev, etc.).​
    
- **Second‑level domain (SLD)**: `aikido` – the name you register.
    
- **Root / main domain**: `aikido.dev` (SLD + TLD together).​
    
- **Subdomain**: `www` in `www.aikido.dev` is a subdomain of `aikido.dev`.​
    

So the hierarchy (right → left) is:  
`dev` (TLD) ← `aikido` (SLD) ← `www` (subdomain).
