
## # üî• What is XSS?

-  **Cross-Site Scripting (XSS)** is a web security vulnerability that allows an attacker to **inject malicious scripts** (usually JavaScript) into a trusted website or web application.  

-  When unsuspecting users load the vulnerable page, their browser executes the injected code as if it came from the legitimate site.

-  This can lead to:

	- Stealing cookies / session tokens (account hijacking)
	    
	- Defacing websites
	    
	- Redirecting victims to malicious pages
	    
	- Keylogging & stealing sensitive data
	    
	- Exploiting browser features or pivoting attacks
