API Documentation

Understand and utilize the FailSafe API.

Token

Get a token for authenticating with the API.

POST /api/token

Parameters:

  • username (string) - The user's username.
  • password (string) - The user's password.

Returns:

  • token (string) - The generated token.

Example:

  • curl -X POST -d '{"username":"alex","password":"alex"}' -H 'Content-Type: application/json' http://<host>/api/token

Vault

List the id & usernames from the vault.

GET /api/vault/:token

Parameters:

  • token (string) - Your authentication token.

Returns:

  • itemid (integer) - ID of the vault item.
  • vaultusername (string) - Username associated with the vault item.
POST /api/vault/:id

Parameters:

  • id (integer) - The ID of the desired vault item.

Body:

  • {"token":"your-token"}

Returns:

  • Details of the specific vault item.
